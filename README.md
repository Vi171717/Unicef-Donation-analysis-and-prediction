# Unicef-Donation-analysis-and-prediction
该项目是统计了自2014年澳大利亚联合国儿童基金会捐赠者的所有捐赠信息，目的是让我们建立预测模型预测捐赠者是否会在首次捐赠六个月内从不定期捐赠转变为定期捐赠，并判断哪些预测者的特征更有可能转变为定期捐赠者。

文件1的代码是一个基础的探索性分析，以确定捐赠者的主要行为和可能性预测其是否会成为定期捐赠者的属性。我结合了捐赠者各行各列的数据制作了多种不同的可视化图片用于揭示所有相关的属性、特征、模式和统计数据，并给出自己的结论。

文件2的代码的探索性分析是文件的进阶版，为了建立更好的模型，我们对业务背景做了进一步的细致裁定和分类，对数据进行更为严谨的清理和整合和分类，同时将筛选具有明显意义的列进行训练。为了让训练的模型拥有精确的预测准度，我们选择了多个模型，进行了多次模型升级并引入超参数，最后通过建立的预测模型对最近半年的捐赠者数据集进行了一次预测模型，通过新的一批捐赠者数据和行为特征判断其在未来六个月转化成定期捐赠者的可能性，在对新的数据集放入kaggle验证准确率和排名。
